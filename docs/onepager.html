GoCalma Challenge Submission

GoCalma

Local-first PDF PII redaction that runs in the browser, not on someone else’s server.

GoCalma is an open-source browser application that detects and redacts sensitive information in PDFs while keeping document contents on the user’s machine. It gives users a safe preprocessing step before sharing any file with AI tools or third-party systems.

Submission Snapshot

97.50% Core text-PDF macro recall on the current repo evaluator
100% Browser-based local processing for the core redaction flow
OCR Supports scanned PDFs and phone-captured document images packaged as PDFs

Problem

People regularly upload sensitive PDFs containing names, addresses, IDs, insurance data, financial details, and medical references into cloud tools. Once those files leave the device, the privacy risk is real.

Solution

GoCalma provides a local workflow: upload a PDF, detect likely PII, review every candidate, export a redacted PDF, and keep an encrypted key file for reversible restoration.

Why It Works

  • Hybrid pipeline: contextual regex, layout heuristics, local NER enrichment, and local OCR.
  • High-recall product posture with human review before final export.
  • Encrypted `.gocalma` package preserves reversible token mapping for un-redaction.
  • No server-side PDF processing required for the submission path.

Real-World Robustness

  • Improved using a wider sample set beyond clean digital PDFs.
  • Includes scanned forms, image-heavy PDFs, and phone-captured documents converted to PDF.
  • OCR stays local in-browser, so the privacy model still holds on messy inputs.
  • Designed for letters, invoices, notices, forms, transcripts, and identity-heavy documents.

Challenge Fit

  • Open-source repository and working prototype are present.
  • Workflow covers upload, detect, review, redact, and download.
  • Encrypted key export supports reversible un-redaction.
  • Core benchmark clears the 90% recall threshold for text-PDF detection.
  • Scanned-document handling is implemented as an extended capability, not just a slide claim.
GoCalma is a credible privacy product, not just a concept demo: strong core recall, browser-only execution, user review before redaction, reversible restoration, and practical handling of scanned and phone-captured PDFs.
